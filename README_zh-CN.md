# Data Insight - 让数据分析变得简单

无需编写任何代码,即可将您的数据转化为洞察。Data Insight 是一款可视化工具包,帮助您通过精美的报告和图表来探索、理解和传达数据发现。

## 您可以做什么?

### 将原始数据转化为理解

上传销售数据、客户信息或任何您想深入了解的数据表格。用日常语言提问,比如"哪些地区表现最好?"或"显示一段时间的趋势",即可获得带有专业可视化的即时答案。

### 从任何地方提取信息

数据被困在图片、截图或文档中?Data Insight 可以提取出数字并将其转换为可分析的格式。非常适合将纸质报告数字化或从演示文稿中提取表格。

### 获得智能推荐

不确定应该使用什么类型的图表?系统会分析您的数据并推荐最有效的可视化方式,解释为什么每个推荐适合您的具体情况。

### 清理和准备数据

自动检测数据中缺失的信息、异常值或不一致之处。获得有用的修复建议,并可选择自动应用修复。

### AI 指导下的探索

告诉系统您的目标,比如"在我的销售数据中找到有趣的模式",然后看着它进行多轮分析,发现您可能错过的趋势、异常和洞察。每个发现都配有清晰的解释和支持性图表。

### 创建专业报告

将所有发现整合成精美、可直接分享的报告。包含多个图表、关键洞察和专业分析 - 全部根据您的数据和问题自动生成。

## 核心功能

### 支持任何数据源

- 电子表格文件(Excel、CSV)
- 数据库信息(MySQL、PostgreSQL、SQLite)
- JSON 数据文件
- 包含表格或图表的图片
- 结构化文本文档
- 带有表格的网页

### 像专家一样分析

- 质量检查,在问题影响您的工作之前发现它们
- 带有易懂解释的统计分析
- 相关性分析,找出数据中的关系
- 趋势检测和模式识别
- 跨不同组别或时间段的比较

### 创建精美的可视化

- 柱状图用于比较
- 折线图用于显示时间趋势
- 散点图用于显示关系
- 面积图用于累积值
- 饼图用于比例
- 热力图用于复杂模式

### 获得即时答案

无需学习公式或编程,只需描述您想要什么:

- "显示收入前 10 名的客户"
- "计算每个季度的平均销售额"
- "找出销售额下降的产品"
- "对比今年和去年"

系统会将您的请求转化为正确的操作,并立即显示结果。

## 工作原理

### 快速分析

1. **加载数据**:选择您的文件或数据源
2. **提出问题**:描述您想了解什么
3. **获得即时结果**:查看带有图表和统计数据的答案
4. **保存报告**:下载专业报告进行分享

### 深度探索

1. **上传数据集**:从任何数据文件开始
2. **设定目标**:告诉系统您想发现什么
3. **观察工作过程**:AI 从多个角度探索您的数据
4. **查看发现**:查看所有发现及其支持证据
5. **分享洞察**:获得可供演示的综合报告

## 可用组件

### 数据管理

**数据加载器**:从电子表格、数据库和文件导入信息。自动理解您的数据结构和类型。

**数据提取器**:从图片、截图、文档和网页中提取表格和数字。将视觉信息转化为可用数据。

**数据质量检查器**:扫描数据中的缺失值、异常值、重复项和不一致之处。建议修复方法,并可自动应用。

### 数据转换

**智能查询工具**:用日常语言提问并获得答案。适用于简单的数据过滤请求或复杂的运行总计计算。

**灵活数据处理器**:执行高级转换,包括百分比、排名、分组和跨数据计算。

### 可视化

**图表推荐器**:分析您的数据特征并推荐最佳可视化方式。解释为什么每个推荐适合您的需求。

**图表生成器**:创建多种风格的专业、可发表的图表。自动格式化坐标轴、标签和图例。

**图表数组构建器**:组合多个可视化,用于全面比较和讲述故事。

### 分析

**统计分析器**:执行常见的统计测试和计算。结果附带对数字含义的通俗解释。

**探索代理**:您的 AI 研究助手,自主探索数据,尝试不同角度,通过多轮分析发现意想不到的洞察。

**报告生成器**:将您的图表、统计数据和关键发现整合成精美的 markdown 报告,可直接分享或发布。

### 完整工作流

**快速分析**:一步式解决方案,加载数据、执行请求的分析、创建合适的图表并生成报告。非常适合快速获得答案。

**智能数据分析**:多轮探索性分析,自动发现洞察。非常适合理解不熟悉的数据集或发现隐藏的模式。

## 获得最佳结果的技巧

**具体明确**:与其说"分析销售",不如说"按地区比较季度销售并识别最佳表现者"。

**从简单开始**:先让基本分析运行起来,然后增加复杂性。快速分析非常适合初步探索。

**首先检查质量**:尽早运行数据质量检查器,在问题影响结论之前发现它们。

**让 AI 探索**:当您不确定要寻找什么时,使用智能数据分析自动发现模式。

**整合洞察**:使用报告生成器将多个分析整合成一个连贯的故事。
